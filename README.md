# FPGA-Based Sign Language Recognition System

## Overview

This project implements a sign language recognition system using an FPGA and flex sensors. The system detects predefined hand gestures and displays the corresponding output on a seven segment diplay.

## Technologies Used

* VHDL
* FPGA Development Board
* Flex Sensors
* Seven Segment Display
* Digital Logic Design

## Features

* Real-time gesture recognition
* Sensor input processing
* LCD output display
* Hardware-based implementation

## Repository Contents

* VHDL source code
* Project report
* System documentation

## Applications

This project aims to improve accessibility and communication by translating sign language gestures into readable text.
